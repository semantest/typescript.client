The Story Behind Semantest

The Problem

Browser Automation Was Broken

We were tired of tests breaking every time a button moved 2 pixels. CSS selectors are fragile. XPath is worse. We needed something better.

The Insight

Semantic Contracts

What if we could describe capabilities instead of implementation details? "Search for products" instead of "#search-btn.primary". A semantic layer that understands intent.

Get Started in Minutes

1. Install

npm install @semantest/browser @semantest/chatgpt.com

2. Configure

import { SemanTestClient } from '@semantest/browser';
const client = new SemanTestClient({ platform: 'chatgpt' });

3. Automate

await client.initialize();
const response = await client.sendPrompt('Generate code');
console.log(response.content);
